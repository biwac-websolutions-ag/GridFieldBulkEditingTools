# GridField bulk editing tools

This module provides a set of tools to enhance [`GridField`](https://docs.silverstripe.org/en/developer_guides/forms/field_types/gridfield/#gridfield), allowing for bulk actions on records. It simplifies managing large sets of data by adding functionalities that are not available in the core `GridField`, saving content editors time and effort.

## Installation

```bash
composer require colymba/gridfield-bulk-editing-tools
```

<https://github.com/silverstripe/silverstripe-gridfield-bulk-editing-tools>
